# Theme Speed-Run

You don't have to settle on the first style you see.

MapVibe has **62+ themes** — and you can switch between all of them on the same city, the same bearing, the same zoom level, in seconds. Mediterranean Vibes. Vintage Noir. And dozens more that haven't been named in a competitor's catalogue.

This is the theme speed-run: finding the one that *feels* like your memory of the place.

---

## Why Theme Matters for Diaspora Prints

The same city at the same angle can feel completely different depending on style:

::: tabs
**Mediterranean Vibes**
Warm ochres and sandy tones. Feels like a summer Sunday. Works beautifully for coastal cities — Rio, Cartagena, Havana.
---
**Vintage Noir**
Deep contrast, aged paper tones. Feels like memory. Works powerfully for dense urban grids — São Paulo's centre, Caracas, Mexico City's historic district.
---
**Minimalist**
Clean lines, monochrome. The cartography itself becomes the art. Works best when the map's geometry is the star — Chicago's grid, Savannah's squares, geometric city layouts.
---
**And 59 more...**
Switch in seconds. No commitment until you're sure.
:::

---

## How to Run the Themes

1. Open [app.mapvibestudio.com](https://app.mapvibestudio.com)
2. Find your location and set your bearing
3. Click through the theme panel — watch the map transform in real time
4. The right one will feel obvious when you see it

::: tip
Don't start with theme. Start with location + bearing. Then run the themes — the right one reveals itself once the geography is right.
:::
